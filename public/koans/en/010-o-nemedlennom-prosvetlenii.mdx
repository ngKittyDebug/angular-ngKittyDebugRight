# Koan №10 — "On Immediate Enlightenment"

<Source>From the oral tradition of the Rinzai school, recorded at Myoshin-ji temple, 1488 CE. Master Toyo spoke it once and never repeated it. Three students remembered it differently. This is the version of the one who wept.</Source>

```js
const a = function(x) { return x * x }
const b = (x) => x * x
```

<Student>"Master, they do the same thing. The arrow is just shorter."</Student>

<Action>The master struck him with a stick — twice.</Action>

<Master>"Identical results — different natures. The crane and the airplane both fly. Will you call them one?"</Master>

<Student>"What is the difference, if the result is the same?"</Student>

<Master>"The first has its own name in the call stack. The second — anonymity. The first has its own this, its own arguments, the right to be a constructor. The second — nothing of its own. It is empty."</Master>

<Student>"Is emptiness weakness?"</Student>

<Action>The master set down the stick.</Action>

<Master>"A monk has no home — therefore the whole world is his home. An arrow function has no this — therefore it is transparent. It will not deceive you with someone else's context. Emptiness is honesty."</Master>

<Haiku>Two paths. One result. Different shadows.</Haiku>

<Question>An arrow function cannot be a constructor, has no arguments object, does not bind this. It renounced everything — and became lighter. But if a tool has renounced capabilities — is that perfection or mutilation? And when you choose syntax — do you choose convenience or a worldview?</Question>
