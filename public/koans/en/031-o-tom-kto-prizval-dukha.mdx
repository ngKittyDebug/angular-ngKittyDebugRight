# Koan №31 — "On the One Who Summoned a Spirit and Did Not Know Its Name"

<Source>Found on the screen of a shattered MacBook in a bin near the gates of Fushimi Inari temple, 2025 CE. Browser history — 47 tabs. In package.json — 130 dependencies. Lines written by a human — zero. Master Kuroda picked up the laptop, read index.js, closed the lid. Said: "There is no author here. There is only desire."</Source>

<Source>The student burst into the hall shouting:</Source>

<Student>"Master! I built an application! In one evening! Authentication, dashboard, REST API, WebSocket, chat, notifications, dark theme!"</Student>

<Master>"Show me."</Master>

<Source>The student opened the laptop. White screen.</Source>

<Student>"One moment... it worked yesterday..."</Student>

<Master>"What broke?"</Master>

<Student>"I don't know."</Student>

<Master>"Read the error."</Master>

```
TypeError: Cannot read properties of undefined (reading 'map')
```

<Student>"It is... somewhere in a component... forty were generated."</Student>

<Master>"What does this hook do?"</Master>

<Student>"I don't know."</Student>

<Master>"Why is it here?"</Master>

<Student>"I don't know."</Student>

<Master>"Whose application is this?"</Master>

<Action>Long silence.</Action>

<Master>"You did not build an application. You asked a spirit to build it for you. The spirit agreed. But the spirit has gone. And you remained — with code you do not understand. You wanted everything at once. You got nothing — gradually."</Master>

<Haiku>A thousand tabs. Not one line your own. The spirit left. The code remains.</Haiku>

<Question>AI gives you code. But not the knowledge of why that code works. The first bug — a snap of fingers, and you wake up in a foreign palace, not knowing where the exit is. If you cannot fix what you "built" — did you build it?</Question>
