# Коан №22 — «О петле, которая солгала»

<Source>Обнаружен в пепле благовоний на алтаре храма Тэнрю-дзи, 1934 г. Пепел лежал кольцами — пять витков. Послушник прочёл их как for. Мастер Сэки прочёл как closure. Оба были правы. Оба ошибались.</Source>

```js
for (var i = 0; i < 5; i++) {
  setTimeout(() => console.log(i), 0)
}
// 5 5 5 5 5
```

<Student>Ученик: «Мастер, я ждал 0 1 2 3 4. Я получил пять пятёрок.»</Student>

<Master>Мастер: «Ты получил правду.»</Master>

<Action>Мастер поставил на стол одну свечу.</Action>

<Master>Мастер: «var — одна свеча. Одна переменная на весь цикл. Пять колбэков смотрят на неё. Когда они наконец откроют глаза — свеча уже догорела до пяти.»</Master>

```js
for (let i = 0; i < 5; i++) {
  setTimeout(() => console.log(i), 0)
}
// 0 1 2 3 4
```

<Master>Мастер: «let — пять свечей. Каждая итерация зажигает свою. var существует в функции. let существует в блоке. У каждой итерации своя жизнь.»</Master>

<Haiku>Пять колбэков. Одна свеча. Пять пятёрок на полу.</Haiku>

<Question>С чем ты уйдёшь: Замыкание закрывает не значение — оно закрывает переменную. Переменная меняется. Ты видишь момент создания или момент исполнения?</Question>
