# Коан №24 — «О async/await и иллюзии прямоты»

Найден в заброшенном телеграфном столбе у дороги к храму Мэйдзи-дзингу, 1961 г. Провод оборван. Сообщение не доставлено. Внутри — свиток: «Сообщение ждало. Столб — нет.»

```js
// путь через .then
fetch('/api')
  .then(r => r.json())
  .then(d => console.log(d))

// путь через await
const d = await fetch('/api')
const j = await d.json()
console.log(j)
```

Ученик: «Мастер, async/await — это просветление. Код стал прямым. Как синхронный.»

Мастер ударил его палкой трижды.

Мастер: «"Как синхронный" — и есть ловушка.»

Мастер взял лист бумаги, сложил его веером — и развернул обратно.

Мастер: «Гладкий лист. Но следы сгибов остались. await не делает код синхронным. Он прячет промис. Под каждым await — точка разрыва. Функция засыпает. Другой код продолжает работу.»

Ученик: «Но разве это плохо?»

Мастер: «Когда ты забываешь про try/catch — плохо. Когда ты await в цикле последовательно там, где можно параллельно — плохо.»

Прямая дорога. Скрытый мост. Ты идёшь — не зная, что летишь.

С чем ты уйдёшь: async/await — синтаксический сахар над промисами. Сахар не меняет природу. Ты знаешь, что происходит под сахаром?
