# Коан №19 — «О строке, которая не строка»

Записан на обёртке от благовоний в монастыре Кофуку-дзи, 1850 г. Благовония сгорели. Обёртка — нет. Мастер сказал: «Содержимое исчезло. Форма осталась. Это шаблонный литерал.»

```js
const name = 'монах'
const greeting = `Приветствую, ${name}`

// Но также:
const html = String.raw`<div class="${name}"></div>`
```

Ученик: «Мастер, это просто удобная строка с подстановкой.»

Мастер: «А если я скажу тебе, что строка — это вызов функции?»

Ученик: «Строка — это функция?»

```js
function zen(strings, ...values) {
  return strings.raw[0] + values[0].toUpperCase()
}

zen`Привет, ${'мир'}`  // 'Привет, МИР'
```

Мастер: «Tagged template. Функция перед обратными кавычками — это тег. Она получает части строки и подстановки отдельно. html\`...\`, css\`...\`, gql\`...\` — это языки внутри языка.»

Ученик: «Строка стала языком...»

Обёртка пуста. Но форма — язык.

С чем ты уйдёшь: Tagged templates превращают строковый литерал в вызов функции. Строка перестаёт быть данными и становится языком внутри языка. Если обратные кавычки могут создать встроенный DSL — где граница между строкой, кодом и языком?
