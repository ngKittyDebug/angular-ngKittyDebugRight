# Коан №40 — «О переводчике, который перевёл себя»

Найден в localStorage браузера на публичном компьютере в библиотеке храма Тофуку-дзи, 2026 г. Ключ — i18n_locale. Значение — 'mu'. Ни один fallback не сработал. Приложение показало пустоту. Библиотекарь сказал: «Наконец-то тихо.»

Ученик пришёл с идеей.

Ученик: «Мастер, я хочу сделать переключатель языка.»

Мастер: «Делай.»

Ученик: «Какую библиотеку взять?»

Мастер: «Ты сказал "хочу сделать". Не "хочу спросить". Сколько языков?»

Ученик: «Два. Русский и английский.»

Мастер: «Тогда зачем библиотека?»

```js
const t = { en: { greeting: 'Hello' }, ru: { greeting: 'Привет' } }
t[locale].greeting
```

Ученик: «Это же... слишком просто.»

Мастер: «Для двух языков и десяти строк — это всё, что нужно. Ты потянулся за большой библиотекой, потому что "так делают". Кто делает? Те, у кого сорок локалей. У тебя — два приветствия. Взял молот для гвоздя.»

Два слова. Один объект. Молоток отложен.

С чем ты уйдёшь: Решение должно соответствовать задаче. Не амбициям. Не тому, "как принято". Ты решаешь задачу — или демонстрируешь знание инструментов?
