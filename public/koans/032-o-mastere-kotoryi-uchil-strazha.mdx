# Коан №32 — «О мастере, который научил стража и ушёл»

Высечен на кремниевой пластине, найденной в фундаменте нового дата-центра у подножья горы Фудзи, 2026 г. Строители приняли её за брак литографии. Инженер прочёл под микроскопом. Расплакался. Написал .cursor/rules. Уволился. Открыл монастырь. В README.md — одна строка: «Линтер видит синтаксис. Ревьювер видит карму.»

Ученик прибежал:

Ученик: «Мастер! Я настроил ИИ-агента для код-ревью! Он проверяет пулл-реквесты!»

Мастер: «Что ты ему сказал?»

Ученик: «"Проверь код на ошибки."»

Мастер: «И что он сказал?»

Ученик: «Сказал — всё хорошо.»

Мастер открыл пулл-реквест и молча показал ученику:

```js
function processData(data) {
  const result = []
  for (let i = 0; i <= data.length; i++) {
    result.push(data[i])
  }
  return result
}
```

Ученик: «Ошибок нет. Код работает... почти.»

Мастер ударил палкой — не ученика. Ноутбук.

Мастер: «Работающий код — это низшая ступень. Отсутствие ошибок — не присутствие качества. Ты дал стражу один глаз и поставил охранять храм с четырьмя воротами.»

Мастер сел и писал весь день. К вечеру передал ученику свиток с детальным промптом для ревьювера — про читаемость, граничные условия, off-by-one, производительность, намерение.

Страж с одним глазом. Четыре ворота. Три открыты.

С чем ты уйдёшь: Идеальный ревьювер-агент — не тот, кто знает все правила. А тот, кого правильно спросили. Промпт — это не инструкция. Это передача взгляда. Если ты не можешь объяснить агенту, что плохой код, — знаешь ли ты сам?
