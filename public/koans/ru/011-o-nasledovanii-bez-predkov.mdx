# Коан №11 — «О наследовании без предков»

<Source>Фрагмент найден в дупле тысячелетней криптомерии на горе Хиэй, 1521 г. Монах Дзякунэн спрятал его от настоятеля, который запрещал говорить о прототипах, называя их «цепями сансары».</Source>

```js
const parent = { wisdom: 42 }
const child = Object.create(parent)

child.wisdom      // 42
child.hasOwnProperty('wisdom')  // false
```

<Student>«Мастер, дитя обладает мудростью. Но говорит, что мудрость — не его.»</Student>

<Master>«Наконец-то честное дитя.»</Master>

<Action>Мастер снял с шеи чётки и повесил на гвоздь. Под гвоздём — полка. Под полкой — стол. Под столом — пол.</Action>

<Master>«Урони бусину. Куда упадёт?»</Master>

<Student>«На полку. Если нет полки — на стол. Если нет стола — на пол.»</Student>

<Master>«Прототипная цепь. Свойство падает вниз, пока не найдёт опору. Если опоры нет — undefined. Бусина в пустоте.»</Master>

<Haiku>Бусина падает. Полка. Стол. Пол. Дальше — небо снизу.</Haiku>

<Question>Объект владеет только тем, что создал сам. Всё остальное — наследие. Ты знаешь, что в тебе твоё, а что — чужое? И в какой момент унаследованное становится собственным?</Question>
