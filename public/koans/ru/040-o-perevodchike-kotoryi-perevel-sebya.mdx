# Коан №40 — «О переводчике, который перевёл себя»

<Source>Найден в localStorage браузера на публичном компьютере в библиотеке храма Тофуку-дзи, 2026 г. Ключ — i18n_locale. Значение — 'mu'. Ни один fallback не сработал. Приложение показало пустоту. Библиотекарь сказал: «Наконец-то тихо.»</Source>

<Source>Ученик пришёл с идеей.</Source>

<Student>«Мастер, я хочу сделать переключатель языка.»</Student>

<Master>«Делай.»</Master>

<Student>«Какую библиотеку взять?»</Student>

<Master>«Ты сказал "хочу сделать". Не "хочу спросить". Сколько языков?»</Master>

<Student>«Два. Русский и английский.»</Student>

<Master>«Тогда зачем библиотека?»</Master>

```js
const t = { en: { greeting: 'Hello' }, ru: { greeting: 'Привет' } }
t[locale].greeting
```

<Student>«Это же... слишком просто.»</Student>

<Master>«Для двух языков и десяти строк — это всё, что нужно. Ты потянулся за большой библиотекой, потому что "так делают". Кто делает? Те, у кого сорок локалей. У тебя — два приветствия. Взял молот для гвоздя.»</Master>

<Haiku>Два слова. Один объект. Молоток отложен.</Haiku>

<Question>Решение должно соответствовать задаче. Не амбициям. Не тому, "как принято". Ты решаешь задачу — или демонстрируешь знание инструментов?</Question>
