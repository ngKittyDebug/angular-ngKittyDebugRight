# Коан №4 — «О потерянном this»

<Source>Из дневников отшельника Фудзивара-но-Скоупу, 1287 г. Горный монастырь Коя-сан. Запись сделана кистью, вымоченной в слезах стажёра.</Source>

```js
const monk = {
  name: 'Кэйсаку',
  speak() { return this.name }
}

const fn = monk.speak
fn()  // undefined
```

<Student>«Мастер! Монах знает своё имя. Но когда я вынул его речь из тела — он забыл, кто он.»</Student>

<Action>Мастер молчал долго.</Action>

<Master>«Вырви сердце из груди воина. Будет ли оно биться за его господина?»</Master>

<Student>«Нет...»</Student>

<Master>«this — не память. Это верность. Она существует только внутри служения. Метод, извлечённый из объекта, — ронин. Он служит window. Или никому.»</Master>

<Student>«Тогда .bind() — это клятва верности?»</Student>

<Action>Мастер слегка кивнул.</Action>

<Master>«А стрелочная функция — это монах, который никогда не покидал храм. У него нет своего this. Он навсегда привязан к месту, где был рождён.»</Master>

<Haiku>Меч без руки. Кому он верен теперь? Лежит в траве.</Haiku>

<Question>Стрелочная функция не теряет this, потому что никогда им не владела. Можно ли потерять то, чего у тебя не было? И что свободнее — тот, кто привязан навечно, или тот, кто привязывается заново при каждом вызове?</Question>
