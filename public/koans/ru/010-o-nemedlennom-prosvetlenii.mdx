# Коан №10 — «О немедленном просветлении»

<Source>Из устной традиции школы Риндзай, записан в храме Мёсин-дзи, 1488 г. Мастер Тоё произнёс его однажды и больше не повторял. Три ученика запомнили по-разному. Это — версия того, кто плакал.</Source>

```js
const a = function(x) { return x * x }
const b = (x) => x * x
```

<Student>«Мастер, они делают одно и то же. Стрелка — просто короче.»</Student>

<Action>Мастер ударил его палкой дважды.</Action>

<Master>«Одинаковый результат — не одинаковая природа. Журавль и самолёт оба летят. Ты назовёшь их одним?»</Master>

<Student>«В чём разница, если результат тот же?»</Student>

<Master>«У первой — своё имя в стеке вызовов. У второй — анонимность. У первой — свой this, свой arguments, право быть конструктором. У второй — ничего своего. Она пуста.»</Master>

<Student>«Пустота — это слабость?»</Student>

<Action>Мастер положил палку.</Action>

<Master>«У монаха нет дома — поэтому весь мир его дом. У стрелочной функции нет this — поэтому она прозрачна. Она не обманет тебя чужим контекстом. Пустота — это честность.»</Master>

<Haiku>Два пути. Один результат. Разные тени.</Haiku>

<Question>Стрелочная функция не может быть конструктором, не имеет arguments, не связывает this. Она отказалась от всего — и стала легче. Но если инструмент отказался от возможностей — это совершенство или увечье? И когда ты выбираешь синтаксис — ты выбираешь удобство или мировоззрение?</Question>
