# Коан №27 — «О цикле, который не перебирает»

<Source>Найден внутри бамбуковой флейты сякухати, принадлежавшей монаху-комусо, деревня Удзи, 2003 г. Флейта не звучала сорок лет. Мастер разрезал бамбук — внутри бумага. На ней — for...in и предупреждение.</Source>

```js
const path = ['камень', 'река', 'храм']

for (const step in path) {
  console.log(step)
}
// '0', '1', '2'
```

<Student>«Мастер, я получил шаги. Но... это строки?»</Student>

<Master>«Ты получил имена, не вещи. for...in даёт ключи. Для него это объект с полями '0', '1', '2'.»</Master>

```js
Array.prototype.curse = 'наследство'
for (const step in path) { console.log(step) }
// '0', '1', '2', 'curse'
```

<Master>«for...in идёт по всем перечислимым свойствам. Своим — и унаследованным. Ты звал только попутчиков. Пришли и мертвецы предков.»</Master>

<Haiku>Флейта молчала сорок лет. Внутри — слова о чужих ключах.</Haiku>

<Question>for...in и for...of звучат похоже. Первый перебирает ключи, второй — значения. Ты знаешь, что нужно тебе?</Question>
