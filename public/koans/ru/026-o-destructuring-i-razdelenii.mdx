# Коан №26 — «О destructuring и разделении единого»

<Source>Записан тушью на рисовом зерне, найденном в ладони статуи Будды Амиды, храм Котоку-ин, Камакура, 1998 г. Реставратор прочёл под лупой. Заплакал. Уволился. Стал фронтенд-разработчиком.</Source>

```js
const universe = { water: { depth: 'бездна', color: 'синий' } }
const { water } = universe
water.color = 'чёрный'

universe.water.color  // 'чёрный'
```
