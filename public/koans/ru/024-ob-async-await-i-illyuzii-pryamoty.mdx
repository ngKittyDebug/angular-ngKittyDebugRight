# Коан №24 — «О async/await и иллюзии прямоты»

<Source>Найден в заброшенном телеграфном столбе у дороги к храму Мэйдзи-дзингу, 1961 г. Провод оборван. Сообщение не доставлено. Внутри — свиток: «Сообщение ждало. Столб — нет.»</Source>

```js
// путь через .then
fetch('/api')
  .then(r => r.json())
  .then(d => console.log(d))

// путь через await
const d = await fetch('/api')
const j = await d.json()
console.log(j)
```

<Student>«Мастер, async/await — это просветление. Код стал прямым. Как синхронный.»</Student>

<Action>Мастер ударил его палкой трижды.</Action>

<Master>«"Как синхронный" — и есть ловушка.»</Master>

<Action>Мастер взял лист бумаги, сложил его веером — и развернул обратно.</Action>

<Master>«Гладкий лист. Но следы сгибов остались. await не делает код синхронным. Он прячет промис. Под каждым await — точка разрыва. Функция засыпает. Другой код продолжает работу.»</Master>

<Student>«Но разве это плохо?»</Student>

<Master>«Когда ты забываешь про try/catch — плохо. Когда ты await в цикле последовательно там, где можно параллельно — плохо.»</Master>
