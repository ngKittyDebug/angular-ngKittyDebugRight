# Коан №32 — «О мастере, который научил стража и ушёл»

<Source>Высечен на кремниевой пластине, найденной в фундаменте нового дата-центра у подножья горы Фудзи, 2026 г. Строители приняли её за брак литографии. Инженер прочёл под микроскопом. Расплакался. Написал .cursor/rules. Уволился. Открыл монастырь. В README.md — одна строка: «Линтер видит синтаксис. Ревьювер видит карму.»</Source>

<Source>Ученик прибежал:</Source>

<Student>«Мастер! Я настроил ИИ-агента для код-ревью! Он проверяет пулл-реквесты!»</Student>

<Master>«Что ты ему сказал?»</Master>

<Student>«"Проверь код на ошибки."»</Student>

<Master>«И что он сказал?»</Master>

<Student>«Сказал — всё хорошо.»</Student>

<Source>Мастер открыл пулл-реквест и молча показал ученику:</Source>

```js
function processData(data) {
  const result = []
  for (let i = 0; i <= data.length; i++) {
    result.push(data[i])
  }
  return result
}
```

<Student>«Ошибок нет. Код работает... почти.»</Student>

<Action>Мастер ударил палкой — не ученика. Ноутбук.</Action>

<Master>«Работающий код — это низшая ступень. Отсутствие ошибок — не присутствие качества. Ты дал стражу один глаз и поставил охранять храм с четырьмя воротами.»</Master>

<Action>Мастер сел и писал весь день. К вечеру передал ученику свиток с детальным промптом для ревьювера — про читаемость, граничные условия, off-by-one, производительность, намерение.</Action>

<Haiku>Страж с одним глазом. Четыре ворота. Три открыты.</Haiku>

<Question>Идеальный ревьювер-агент — не тот, кто знает все правила. А тот, кого правильно спросили. Промпт — это не инструкция. Это передача взгляда. Если ты не можешь объяснить агенту, что плохой код, — знаешь ли ты сам?</Question>
