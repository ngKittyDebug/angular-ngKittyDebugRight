# Коан №17 — «О наблюдателе, который изменяет»

<Source>Найден за зеркалом в покоях настоятеля монастыря Дайго-дзи, 1756 г. Зеркало вешали в 1603 году. За ним — свиток и записка: «Смотришь в зеркало — зеркало смотрит в тебя. Proxy.»</Source>

```js
const monk = { enlightened: false }

const observed = new Proxy(monk, {
  get(target, key) {
    console.log(`кто-то читает ${key}`)
    return target[key]
  }
})

observed.enlightened // 'кто-то читает enlightened'
```

<Student>«Мастер, я создал объект, который знает о каждом прикосновении.»</Student>

<Master>«Он знает. Но изменился ли он от того, что за ним наблюдают?»</Master>

<Student>«Нет. Значения те же.»</Student>

<Master>«В физике — нет. В квантовой — наблюдение меняет результат. Proxy позволяет тебе быть квантовым наблюдателем. Ты можешь не только видеть — ты можешь перехватить, изменить, запретить.»</Master>

<Student>«Vue 3 построен на Proxy?»</Student>

<Master>«И Angular Signals. И MobX. Реактивность — это Proxy с намерением.»</Master>

<Haiku>Зеркало смотрит. Монах меняется. Или нет?</Haiku>

<Question>Vue 3 построен на Proxy. Каждое реактивное свойство — под наблюдением. Если фреймворк наблюдает за каждым прикосновением к данным, — остаются ли данные свободными? И существует ли в реактивном мире объект, на который никто не смотрит?</Question>
