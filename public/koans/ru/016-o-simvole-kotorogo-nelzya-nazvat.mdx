# Коан №16 — «О символе, которого нельзя назвать»

<Source>Найден внутри бронзового колокола храма Тисон-ин, 1711 г. Колокол звучал фальшиво сорок лет. Когда его переплавили — нашли свиток. Мастер заметил: «Правильное место для имени, которое нельзя повторить.»</Source>

```js
const s = Symbol('koan')
const t = Symbol('koan')

s === t   // false
s.toString()  // 'Symbol(koan)'
```

<Student>«Мастер, я дал им одинаковое имя. Почему они не равны?»</Student>

<Action>Мастер указал на двух монахов в одинаковых одеждах.</Action>

<Master>«Оба носят серое. Оба бриты. Оба названы "монах". Они — один человек?»</Master>

<Student>«Нет, но...»</Student>

<Master>«Описание — не сущность. 'koan' — это записка, привязанная к вороту. Каждый Symbol() рождается единственным. Как каждый вдох.»</Master>

<Student>«Зачем нужна уникальность без возможности сравнить?»</Student>

<Action>Мастер вынул из кармана ключ без надписи.</Action>

<Master>«Этот ключ открывает один замок. На ключе нет имени. На замке нет имени. Но они знают друг друга. Symbol — это ключ. Не имя.»</Master>

<Haiku>Два монаха. Одно платье. Разные судьбы.</Haiku>

<Question>Строка — общественна. Любой может написать то же имя и получить доступ. Символ — приватен по природе. Если два ключа выглядят одинаково, но открывают разные двери — что определяет идентичность: внешний вид или происхождение?</Question>
