# Коан №9 — «О двух ничто»

Из переписки монахов монастыря Дайтоку-дзи, 1456 г. Младший монах спросил старшего: «Если пустота одна — зачем у неё два имени?» Старший сжёг письмо. Это — реконструкция.

```js
let a = undefined
let b = null

a == b    // true
a === b   // false
```

Ученик положил перед мастером два камня.

Ученик: «Мастер, они равны — но не идентичны. Как?»

Мастер указал на два пустых стула.

Мастер: «На этом стуле никто не сидит, потому что никто не пришёл. На том — никто не сидит, потому что я убрал гостя.»

Ученик: «Оба пусты!»

Мастер: «Оба пусты. Но первый не знал присутствия. Второй — помнит отсутствие.»

Ученик: «undefined — это "не было". null — это "было, но убрано"?»

Мастер взял оба камня и сжал в кулаке.

Мастер: «Спроси движок — они похожи. Спроси программиста — они различны. Спроси философа — они оба страдание.»

Два пустых следа. Один — не приходил. Другой — ушёл.

С чем ты уйдёшь: undefined назначает сам язык — это его молчание. null назначает программист — это его решение. Но если оба означают «ничего» — кто из них честнее? Тот, кто молчит случайно, или тот, кто молчит намеренно?
