# Коан №9 — «О двух ничто»

<Source>Из переписки монахов монастыря Дайтоку-дзи, 1456 г. Младший монах спросил старшего: «Если пустота одна — зачем у неё два имени?» Старший сжёг письмо. Это — реконструкция.</Source>

```js
let a = undefined
let b = null

a == b    // true
a === b   // false
```

<Action>Ученик положил перед мастером два камня.</Action>

<Student>Ученик: «Мастер, они равны — но не идентичны. Как?»</Student>

<Action>Мастер указал на два пустых стула.</Action>

<Master>Мастер: «На этом стуле никто не сидит, потому что никто не пришёл. На том — никто не сидит, потому что я убрал гостя.»</Master>

<Student>Ученик: «Оба пусты!»</Student>

<Master>Мастер: «Оба пусты. Но первый не знал присутствия. Второй — помнит отсутствие.»</Master>

<Student>Ученик: «undefined — это "не было". null — это "было, но убрано"?»</Student>

<Action>Мастер взял оба камня и сжал в кулаке.</Action>

<Master>Мастер: «Спроси движок — они похожи. Спроси программиста — они различны. Спроси философа — они оба страдание.»</Master>

<Haiku>Два пустых следа. Один — не приходил. Другой — ушёл.</Haiku>

<Question>С чем ты уйдёшь: undefined назначает сам язык — это его молчание. null назначает программист — это его решение. Но если оба означают «ничего» — кто из них честнее? Тот, кто молчит случайно, или тот, кто молчит намеренно?</Question>
