# Коан №10 — «О немедленном просветлении»

Из устной традиции школы Риндзай, записан в храме Мёсин-дзи, 1488 г. Мастер Тоё произнёс его однажды и больше не повторял. Три ученика запомнили по-разному. Это — версия того, кто плакал.

```js
const a = function(x) { return x * x }
const b = (x) => x * x
```

Ученик: «Мастер, они делают одно и то же. Стрелка — просто короче.»

Мастер ударил его палкой дважды.

Мастер: «Одинаковый результат — не одинаковая природа. Журавль и самолёт оба летят. Ты назовёшь их одним?»

Ученик: «В чём разница, если результат тот же?»

Мастер: «У первой — своё имя в стеке вызовов. У второй — анонимность. У первой — свой this, свой arguments, право быть конструктором. У второй — ничего своего. Она пуста.»

Ученик: «Пустота — это слабость?»

Мастер положил палку.

Мастер: «У монаха нет дома — поэтому весь мир его дом. У стрелочной функции нет this — поэтому она прозрачна. Она не обманет тебя чужим контекстом. Пустота — это честность.»

Два пути. Один результат. Разные тени.

С чем ты уйдёшь: Стрелочная функция не может быть конструктором, не имеет arguments, не связывает this. Она отказалась от всего — и стала легче. Но если инструмент отказался от возможностей — это совершенство или увечье? И когда ты выбираешь синтаксис — ты выбираешь удобство или мировоззрение?
