# Коан №23 — «О NaN, который не равен себе»

<Source>Вырезан на дне медной чаши для подаяний, храм Кэнтё-дзи, 1953 г. Чашу передавали из рук в руки триста лет. Надпись нашли, когда чаша проржавела насквозь. Мастер Асахина сказал: «Наконец-то она стала прозрачной.»</Source>

```js
NaN === NaN   // false
```

<Action>Долгое молчание.</Action>

<Master>«NaN — это не значение. Это признание провала. parseInt('камень'). Math.sqrt(-1). Ты попросил число — а мир не смог ответить. NaN — это шрам там, где число должно было быть.»</Master>

<Student>«Как проверить, что это NaN?»</Student>

<Master>«Number.isNaN(). Не голый isNaN() — он лжёт. Только Number.isNaN() говорит правду.»</Master>

<Haiku>Вещь без имени. Не равна себе. Есть — и нет.</Haiku>
