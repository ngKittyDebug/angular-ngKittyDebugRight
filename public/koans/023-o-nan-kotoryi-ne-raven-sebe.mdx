# Коан №23 — «О NaN, который не равен себе»

Вырезан на дне медной чаши для подаяний, храм Кэнтё-дзи, 1953 г. Чашу передавали из рук в руки триста лет. Надпись нашли, когда чаша проржавела насквозь. Мастер Асахина сказал: «Наконец-то она стала прозрачной.»

```js
NaN === NaN   // false
```

Мастер ударил ученика палкой.

Мастер: «Почему не спрашиваешь?»

Ученик: «Потому что любой вопрос об этом будет глупым.»

Мастер: «Наконец мудрость.»

Долгое молчание.

Мастер: «NaN — это не значение. Это признание провала. parseInt('камень'). Math.sqrt(-1). Ты попросил число — а мир не смог ответить. NaN — это шрам там, где число должно было быть.»

Ученик: «Как проверить, что это NaN?»

Мастер: «Number.isNaN(). Не голый isNaN() — он лжёт. Только Number.isNaN() говорит правду.»

Вещь без имени. Не равна себе. Есть — и нет.

С чем ты уйдёшь: NaN — единственная сущность в JS, которая отрицает собственное существование. Когда ты в последний раз проверял результат математической операции?
