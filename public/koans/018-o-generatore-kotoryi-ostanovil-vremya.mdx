# Коан №18 — «О генераторе, который остановил время»

Найден в механических часах монастыря Эннён-дзи, 1801 г. Часы остановились. Мастер разобрал механизм — внутри свиток, намотанный на пружину. Часы шли на этом свитке восемьдесят лет.

```js
function* monastery() {
  yield 'утро'
  yield 'день'
  yield 'вечер'
}

const day = monastery()
day.next() // { value: 'утро', done: false }
day.next() // { value: 'день', done: false }
```

Ученик: «Мастер, функция остановилась на полуслове. И ждёт.»

Мастер: «Она не остановилась. Она отдала управление.»

Ученик: «Но она заморожена!»

Мастер: «Снег заморожен. Генератор — на паузе. Разница: снег не ждёт твоего .next(). Генератор — ждёт.»

Ученик: «Кто решает, когда продолжить?»

Мастер: «Тот, кто держит итератор. Не функция. Функция отдала волю — добровольно.»

Пружина намотана. Часы ждут. Время — у того, кто держит ключ.

С чем ты уйдёшь: Генератор отдаёт контроль наружу. Он не решает, когда продолжить — это решает тот, кто держит итератор. Если функция отдала свою волю вызывающему — это доверие или беспомощность? И сколько генераторов в твоём коде замерли навсегда, потому что никто не позвал .next()?
