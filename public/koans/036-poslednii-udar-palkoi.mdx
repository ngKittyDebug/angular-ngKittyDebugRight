# Коан №36 — «Последний удар палкой»

Не записан нигде. Не найден никем. Этот коан мастер не диктовал — он молчал. Ученик записал тишину. Мастер прочёл запись и сказал: «Почти.»

Ученик пришёл однажды без вопроса.

Мастер ждал вопроса. Ученик молчал. Мастер ждал ещё. Ученик смотрел в сад.

Мастер поднял палку.

Ученик не отвернулся.

Мастер опустил палку.

Мастер: «Ты перестал спрашивать.»

Ученик: «Я перестал ждать ответа.»

Мастер встал и ушёл в комнату. Через минуту вернулся с чаем — для ученика.

Сад. Тишина. Палка опущена. Чай — горячий.

С чем ты уйдёшь: yield завершён. Генератор вернул `{ value: undefined, done: true }`. Дальше — пустой файл, мигающий курсор и тишина, в которой рождается твой собственный код. Если ты не можешь объяснить JavaScript себе — ты его не знаешь. Если можешь — тебе не нужен мастер.
