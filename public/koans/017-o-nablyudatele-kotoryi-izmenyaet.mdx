# Коан №17 — «О наблюдателе, который изменяет»

Найден за зеркалом в покоях настоятеля монастыря Дайго-дзи, 1756 г. Зеркало вешали в 1603 году. За ним — свиток и записка: «Смотришь в зеркало — зеркало смотрит в тебя. Proxy.»

```js
const monk = { enlightened: false }

const observed = new Proxy(monk, {
  get(target, key) {
    console.log(`кто-то читает ${key}`)
    return target[key]
  }
})

observed.enlightened // 'кто-то читает enlightened'
```

Ученик: «Мастер, я создал объект, который знает о каждом прикосновении.»

Мастер: «Он знает. Но изменился ли он от того, что за ним наблюдают?»

Ученик: «Нет. Значения те же.»

Мастер: «В физике — нет. В квантовой — наблюдение меняет результат. Proxy позволяет тебе быть квантовым наблюдателем. Ты можешь не только видеть — ты можешь перехватить, изменить, запретить.»

Ученик: «Vue 3 построен на Proxy?»

Мастер: «И Angular Signals. И MobX. Реактивность — это Proxy с намерением.»

Зеркало смотрит. Монах меняется. Или нет?

С чем ты уйдёшь: Vue 3 построен на Proxy. Каждое реактивное свойство — под наблюдением. Если фреймворк наблюдает за каждым прикосновением к данным, — остаются ли данные свободными? И существует ли в реактивном мире объект, на который никто не смотрит?
