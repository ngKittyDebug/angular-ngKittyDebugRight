# Коан №4 — «О потерянном this»

Из дневников отшельника Фудзивара-но-Скоупу, 1287 г. Горный монастырь Коя-сан. Запись сделана кистью, вымоченной в слезах стажёра.

```js
const monk = {
  name: 'Кэйсаку',
  speak() { return this.name }
}

const fn = monk.speak
fn()  // undefined
```

Ученик: «Мастер! Монах знает своё имя. Но когда я вынул его речь из тела — он забыл, кто он.»

Мастер молчал долго.

Мастер: «Вырви сердце из груди воина. Будет ли оно биться за его господина?»

Ученик: «Нет...»

Мастер: «this — не память. Это верность. Она существует только внутри служения. Метод, извлечённый из объекта, — ронин. Он служит window. Или никому.»

Ученик: «Тогда .bind() — это клятва верности?»

Мастер слегка кивнул.

Мастер: «А стрелочная функция — это монах, который никогда не покидал храм. У него нет своего this. Он навсегда привязан к месту, где был рождён.»

Меч без руки. Кому он верен теперь? Лежит в траве.

С чем ты уйдёшь: Стрелочная функция не теряет this, потому что никогда им не владела. Можно ли потерять то, чего у тебя не было? И что свободнее — тот, кто привязан навечно, или тот, кто привязывается заново при каждом вызове?
