# Коан №29 — «О лжи восьми ног»

Извлечён из панциря краба, выловленного в заливе Мацусима, 2017 г. Рыбак отнёс его в храм Дзуйган-дзи. Настоятель прочёл. Побледнел. Сказал: «Краб знал JavaScript.» Краба отпустили.

```js
const falsies = [0, '', null, undefined, NaN, false, -0, 0n]
falsies.filter(Boolean)  // []
```

Ученик: «Мастер, я собрал восемь значений. Все — ложь.»

Мастер поднял пустую чашу.

Мастер: «Чаша существует. Чаша пуста. Она ложна?»

Ученик молчал.

Мастер: «Ноль — это число, означающее "ничего". Пустая строка — строка, которая молчит. JavaScript не различает "пустое" и "ложное". Для него молчание — ложь. Но ты — человек. Ты должен различать.»

Восемь крабовых ног. Ни одна не несёт истины. Краб идёт.

С чем ты уйдёшь: Восемь falsy-значений — не ложь. Они пусты, нулевы, неопределены. JavaScript назвал их ложью, потому что ему нужна была бинарность. Если пустая чаша — это false, — описывает ли система типов реальность, или создаёт собственную?
