# Коан №29 — «О лжи восьми ног»

<Source>Извлечён из панциря краба, выловленного в заливе Мацусима, 2017 г. Рыбак отнёс его в храм Дзуйган-дзи. Настоятель прочёл. Побледнел. Сказал: «Краб знал JavaScript.» Краба отпустили.</Source>

```js
const falsies = [0, '', null, undefined, NaN, false, -0, 0n]
falsies.filter(Boolean)  // []
```

<Student>Ученик: «Мастер, я собрал восемь значений. Все — ложь.»</Student>

<Action>Мастер поднял пустую чашу.</Action>

<Master>Мастер: «Чаша существует. Чаша пуста. Она ложна?»</Master>

<Action>Ученик молчал.</Action>

<Master>Мастер: «Ноль — это число, означающее "ничего". Пустая строка — строка, которая молчит. JavaScript не различает "пустое" и "ложное". Для него молчание — ложь. Но ты — человек. Ты должен различать.»</Master>

<Haiku>Восемь крабовых ног. Ни одна не несёт истины. Краб идёт.</Haiku>

<Question>С чем ты уйдёшь: Восемь falsy-значений — не ложь. Они пусты, нулевы, неопределены. JavaScript назвал их ложью, потому что ему нужна была бинарность. Если пустая чаша — это false, — описывает ли система типов реальность, или создаёт собственную?</Question>
