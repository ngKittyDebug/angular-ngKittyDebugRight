# Коан №2 — «О равенстве неравных»

<Source>Из обгоревшего пергамента храма Энряку-дзи, 1231 г. Приписывается монаху Рэндану, толкователю Абстрактных Операций.</Source>

```js
[] == ![]   // true
```

<Student>Ученик: «Мастер, пустой массив равен отрицанию пустого массива. Это... невозможно.»</Student>

<Master>Мастер: «Ты спросил у языка "равны ли они?". Язык начал работу по Абстрактному Алгоритму Сравнения. Он не обязан думать о смысле.»</Master>

<Student>Ученик: «Но [] — это объект. ![] — это false. Как они могут быть равны?»</Student>

<Master>Мастер: «Потому что ты написал ==. Не ===. Ты попросил сравнения с принуждением. Язык принудил. Не жалуйся на результат принуждения, которое сам заказал.»</Master>

<Student>Ученик: «Тогда === всегда лучше?»</Student>

<Action>Мастер закрыл глаза.</Action>

<Master>Мастер: «Всегда — слово самонадеянных. Но да.»</Master>

<Haiku>Пустая чаша. Отрицание пустой чаши. Оба — на столе.</Haiku>

<Question>С чем ты уйдёшь: Язык исполняет правила, не намерения. Когда ты пишешь ==, ты подписываешь договор с Абстрактным Алгоритмом. Ты читал его?</Question>
